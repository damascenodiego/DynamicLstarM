LogicProcessor package is an Eclipse IDE project
that can be imported and run on Ubuntu 13.0+ 64 bits
and Java version 8. 

There are several files for implementation, test,
and experiments. You should start on 
test.br.usp.icmc.test.fsm.reader.FFSMReaderTest for
some tests on FFSM properties. The implementation is 
under src folder. Also there are some FeatureIDE files
for feature models and UMLetino for FSM/FFSMs.

R_data Folder have some information about experiments
